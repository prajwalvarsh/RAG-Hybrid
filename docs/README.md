# RAG Hybrid Search

A production-grade Retrieval-Augmented Generation (RAG) system that answers questions from a document corpus using hybrid search, reranking, and grounded generation with citation verification.

## Features

- **Hybrid Search**: Combines dense vector retrieval (OpenAI embeddings) with sparse keyword search (BM25)
- **Reciprocal Rank Fusion**: Intelligently merges results from both search methods
- **Reranking**: Improves result quality before generation
- **Grounded Generation**: Generates answers with verified citations
- **Evaluation Metrics**: Built-in RAGAS evaluation suite
